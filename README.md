📄 AI Resume Screening System

An intelligent AI-powered Resume Screening System that automates candidate shortlisting by analyzing resumes and matching them with job descriptions using Natural Language Processing (NLP) and Machine Learning techniques.

This project helps HR professionals reduce manual effort and quickly identify the most suitable candidates based on skills, experience, and relevance.

🚀 Features
📑 Extracts key information from resumes:
Skills
Experience
Education
Certifications
🤖 Uses TF-IDF Vectorization to convert text into numerical form
📊 Applies Cosine Similarity to match resumes with job descriptions
📈 Ranks candidates based on relevance score
⚡ Automates candidate shortlisting process
🎯 Improves hiring efficiency and reduces bias
🛠️ Tools & Technologies Used

This project was built using the following tools and technologies:

Python – Core programming language
Jupyter Notebook / Google Colab – Development environment
Scikit-learn – TF-IDF Vectorizer & similarity calculations
Pandas – Data handling and preprocessing
NumPy – Numerical computations
Natural Language Processing (NLP) techniques
🧠 How It Works
Upload resumes and job description
Preprocess text (cleaning, tokenization, etc.)
Convert text into vectors using TF-IDF
Compute similarity between resumes and job description
Rank resumes based on similarity scores
Output top candidates
